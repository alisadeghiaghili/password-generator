Anwendungsfälle

Praxisbeispiele für die Verwendung in Anwendungen, Skripten und Workflows.

1. Passwortspeicherung in Anwendungen

Python
from password_generator import generate
for name in ["email", "bank", "social"]:
    print(f"{name}: {generate(length=20)}")

2. Verschlüsselungs-Passphrasen

Python
from password_generator import generate_passphrase
phrase = generate_passphrase(words=6, capitalize=True, separator=" ")
print(f"Verschlüsselungsschlüssel: {phrase}")

3. 2FA-Backup-Codes

Python
from password_generator import generate_pin
for i in range(10):
    print(f"Code {i+1}: {generate_pin(length=6, avoid_repeats=True)}")

4. DevOps-Infrastruktur

Python
from password_generator import generate, generate_pin
config = {
    "db_password": generate(length=32, exclude_ambiguous=True),
    "api_key": generate(length=48, symbols=False),
    "admin_pin": generate_pin(length=8, avoid_sequential=True),
}

5. Passwort-Stärke-Audit

Python
from password_generator import analyze
for pwd in ["password123", "Tr0ub4dor&3", "k8#Qx!2mNp@Lw9Rz"]:
    r = analyze(pwd)
    print(f"[Score {r.score}] {r.feedback[0] if r.feedback else 'OK'}")

6. Passwort-Manager-Export

Python
from password_generator import generate
import json
entries = [{"service": s, "password": generate(length=20)} for s in ["github", "aws"]]
with open("passwords.json", "w") as f:
    json.dump(entries, f, indent=2)

7. CI/CD-Secrets

bash
export SECRET_KEY=$(python cli.py --length 48 --json | jq -r '.passwords[0]')

8. Sichere Clipboard-Workflow

Python
from password_generator import generate, analyze, copy_to_clipboard
pwd = generate(length=24)
if analyze(pwd).score >= 3:
    copy_to_clipboard(pwd, auto_clear_seconds=30)

9. Benutzerdefinierte Wortliste

Python
from password_generator import generate_passphrase, PassphraseConfig
config = PassphraseConfig(words=4, wordlist_path="meine_wortliste.txt")
print(generate_passphrase(config))

10. Stapelgenerierung mit Bericht

Python
from password_generator import generate, analyze
results = [(generate(length=20), analyze(generate(length=20))) for _ in range(10)]
avg = sum(r.score for _, r in results) / len(results)
print(f"Durchschnittliche Bewertung: {avg:.1f}/4")
