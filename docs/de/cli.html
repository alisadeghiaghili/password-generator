CLI-Anleitung

Interaktiver Assistent für technische Laien und leistungsstarke Kommandozeilen-Argumente für Skripting und Automatisierung. Mit rich installiert, zeigt der CLI farbige Panels, Stärkebalken und formatierte Tabellen.

Rich CLI: Installieren mit pip install password-generator[cli] für schöne Terminalausgabe mit farbigen Panels, Stärkebalken und formatierten Tabellen.

Interaktiver Modus

bash
python cli.py

Passwörter generieren

bash
python cli.py --length 16
python cli.py --length 20 --count 5
python cli.py --length 12 --no-lowercase --no-symbols
python cli.py --length 20 --exclude-ambiguous

Passphrasen generieren

bash
python cli.py --passphrase
python cli.py --passphrase --words 6 --separator " " --capitalize

PINs generieren

bash
python cli.py --pin
python cli.py --pin --pin-length 6
python cli.py --pin --pin-length 8 --avoid-repeats --avoid-sequential

Analysieren

bash
python cli.py --analyze "MyP@ssw0rd"
python cli.py --analyze "Tr0ub4dor&3" --json

JSON-Ausgabe

bash
python cli.py --length 20 --json
python cli.py --passphrase --words 4 --json
