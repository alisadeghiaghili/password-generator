Willkommen

Ein sicheres, konfigurierbares Passwort-Generierungs-Toolkit für Python. Erstellen Sie zufällige Passwörter, XKCD-style Passphrasen, numerische PINs und analysieren Sie Passwortstärke — alles über eine einfache API oder ein interaktives CLI.

🔐

Zufällige Passwörter

Kryptographisch sichere Passwörter mit vollständig anpassbaren Zeichensätzen. Länge von 4 bis 256 Zeichen.

🐴

Passphrasen

XKCD-style einprägsame Passphrasen aus einer 2048-Wörter-Liste. "correct-horse-battery-staple" Stil.

🔢

PINs

Numerische Codes mit Wiederholungs- und Sequenzvermeidung. Perfekt für 2FA-Backup-Codes.

📊

Stärkeanalyse

Entropie-Bewertung, Knackzeit-Schätzungen, Mustererkennung. Wissen Sie, wie stark Ihre Passwörter wirklich sind.

📋

Zwischenablage

Automatisches Kopieren mit zeitgesteuerter Löschung. Plattformübergreifende Unterstützung.

⌨️

Interaktives CLI

Schöne Rich Terminal UI mit farbigen Panels und Stärkebalken. JSON-Ausgabe für Skripting und Automatisierung.

Schnellstart

Python
from password_generator import generate, generate_passphrase, generate_pin, analyze

# Ein zufälliges 20-Zeichen-Passwort generieren
password = generate(length=20)
print(password)  # z.B. "k8#Qx!2mNp@Lw9Rz"

# Eine 5-Wörter-Passphrase generieren
passphrase = generate_passphrase(words=5, separator=" ", capitalize=True)
print(passphrase)  # z.B. "Correct Horse Battery Staple Moon"

# Eine 6-stellige PIN generieren
pin = generate_pin(length=6, avoid_repeats=True, avoid_sequential=True)
print(pin)  # z.B. "849205"

# Passwortstärke analysieren
report = analyze("P@ssw0rd123")
print(f"Bewertung: {report.score}/4")  # 0=Sehr schwach ... 4=Sehr stark
Keine Abhängigkeiten — Verwendet nur Pythons Standardbibliothek (secrets, string, math). Keine externen Pakete erforderlich.
