Installation

Anforderungen

  • Python 3.10 oder höher
  • Keine externen Abhängigkeiten — verwendet nur Pythons Standardbibliothek
  • Linux-Zwischenablage erfordert xclip oder xsel

Über PyPI

bash
pip install password-generator

Aus Quellcode

bash
git clone https://github.com/alisadeghiaghili/password-generator.git
cd password-generator
pip install -e .

Entwicklungsinstallation

bash
git clone https://github.com/alisadeghiaghili/password-generator.git
cd password-generator
python -m venv venv
source venv/bin/activate  # macOS/Linux
venv\Scripts\activate     # Windows
pip install -e ".[dev]"
pytest

Überprüfung

Python
from password_generator import generate
print(generate(length=16))
Tipp: Für Clipboard-Unterstützung unter Linux installieren Sie xclip: sudo apt install xclip
