Stärkeanalyse

Analysieren Sie Passwortstärke mit Entropie-Bewertung, Knackzeit-Schätzungen und Mustererkennung.

Grundlegende Verwendung

Python
from password_generator import analyze
report = analyze("MyP@ssw0rd123!")
print(f"Bewertung: {report.score}/4")
print(f"Entropie: {report.entropy:.0f} Bits")

Bewertung verstehen

WertBeschriftungVersucheBeschreibung
0Sehr schwach< 1.000Sofort geknackt
1Schwach< 1.000.000Minuten bis Stunden
2Fair< 100.000.000Tage bis Monate
3Stark< 10.000.000.000Jahrzehnte
4Sehr stark≥ 10.000.000.000Jahrhunderte+

Knackzeit-Schätzungen

SzenarioGeschwindigkeitBeschreibung

offline_slow_hashing10K/SekundeGestohlener Hash, langsamer Algorithmus
offline_fast_hashing10B/SekundeGestohlener Hash, schneller Algorithmus

Vollständiger Bericht

Python
report = analyze("k8#Qx!2mNp@Lw9Rz")
print(report.password)         # "****************" (maskiert)
print(report.score)            # 4
print(report.entropy)          # ~106
print(report.crack_times)      # dict mit Szenario → Zeit
print(report.feedback)         # ["Good password!"]
print(report.patterns)         # [] (keine schwachen Muster)
Sicherheitshinweis: Das Passwort wird im Bericht maskiert (als Sterne) um eine versehentliche Offenlegung zu verhindern.
