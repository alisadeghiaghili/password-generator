Zwischenablage

Plattformübergreifende Clipboard-Unterstützung mit automatischer Löschung. Kopieren Sie Passwörter sicher — sie verschwinden nach einer konfigurierbaren Zeitspanne.

Grundlegende Verwendung

Python
from password_generator import generate, copy_to_clipboard
password = generate(length=24)
copy_to_clipboard(password)

Automatische Löschung

Python
copy_to_clipboard(password, auto_clear_seconds=30)   # 30 Sekunden
copy_to_clipboard(password, auto_clear_seconds=120)  # 2 Minuten

Manuelles Löschen

Python
from password_generator import clear_clipboard
clear_clipboard()
