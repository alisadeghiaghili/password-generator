Zufällige Passwörter

Generieren Sie kryptographisch sichere zufällige Passwörter mit vollständiger Kontrolle über Zeichensätze, Länge und Mehrdeutigkeitsfilterung.

Grundlegende Verwendung

Python
from password_generator import generate

# Standard: 16 Zeichen, alle Zeichentypen
password = generate()
print(password)  # z.B. "k8#Qx!2mNp@Lw9Rz"

Benutzerdefinierte Länge

Python
short = generate(length=8)   # Minimum 4
long = generate(length=64)   # Maximum 256

Zeichensätze steuern

Python
generate(digits=False, symbols=False)        # Nur Buchstaben
generate(symbols=False)                      # Alphanumerisch
generate(lowercase=False, symbols=False)     # Großbuchstaben + Ziffern

Benutzerdefinierte Symbole

Python
generate(symbol_chars="!@#$%")               # Nur bestimmte Symbole
generate(symbol_chars="-_.~")                # Shell-sichere Symbole

Mehrdeutige Zeichen ausschließen

Zeichen wie l, I, 1, O, 0 sehen in manchen Schriftarten ähnlich aus:

Python
generate(length=20, exclude_ambiguous=True)  # Kein: l, I, 1, O, 0, o

GeneratorConfig verwenden

Python
from password_generator import generate, GeneratorConfig

config = GeneratorConfig(length=24, exclude_ambiguous=True)
for i in range(5):
    print(generate(config))

Parameter

ParameterTypStandardBeschreibung
lengthint16Passwortlänge (4–256)
uppercaseboolTrueA–Z einschließen
lowercaseboolTruea–z einschließen
digitsboolTrue0–9 einschließen
symbolsboolTrueSymbole einschließen
symbol_charsstr!@#$%^&*…Benutzerdefinierte Symbole
exclude_ambiguousboolFalsel, I, 1, O, 0, o entfernen

So funktioniert es

  1. Pool-Seeding — Ein Zeichen aus jedem aktivierten Pool garantiert Vielfalt
  2. Auffüllen — Restliche Zeichen aus kombiniertem Pool
  3. Fisher-Yates-Shuffle — Kryptographisch sicher für gleichmäßige Verteilung
Sicherheit: Verwendet Pythons secrets-Modul. Verwenden Sie niemals random für Passwörter.
