PINs

Generieren Sie zufällige numerische PINs mit optionaler Wiederholungs- und Sequenzvermeidung. Perfekt für 2FA-Backup-Codes.

Grundlegende Verwendung

Python
from password_generator import generate_pin
pin = generate_pin()  # Standard: 4-stellig
print(pin)  # z.B. "8492"

Benutzerdefinierte Länge

Python
generate_pin(length=6)  # 6-stellig
generate_pin(length=8)  # 8-stellig

Wiederholungen vermeiden

Python
generate_pin(length=6, avoid_repeats=True)
# Nie: 111234, 999888

Sequenzen vermeiden

Python
generate_pin(length=4, avoid_sequential=True)
# Nie: 1234, 4321, 5678

Beide Einschränkungen kombinieren

Python
generate_pin(length=6, avoid_repeats=True, avoid_sequential=True)

Parameter

ParameterTypStandardBeschreibung
lengthint4PIN-Länge (1–12)
avoid_repeatsboolFalse3+ identische Ziffern vermeiden
avoid_sequentialboolFalseAuf-/absteigende Sequenzen vermeiden

Mögliche Kombinationen

LängeMögliche PINsEntropie

Warnung: PINs haben begrenzte Entropie. Verwenden Sie 6+ Ziffern für sensible Anwendungen.
