Passphrasen

Generieren Sie einprägsame XKCD-style Passphrasen aus einer 2048-Wörter-Liste. Leicht zu merken, schwer zu knacken.

Grundlegende Verwendung

Python
from password_generator import generate_passphrase

# Standard: 4 Wörter, mit Bindestrich getrennt
passphrase = generate_passphrase()
print(passphrase)  # z.B. "correct-horse-battery-staple"

Benutzerdefinierte Wortanzahl

Python
generate_passphrase(words=3)   # ~33 Bits
generate_passphrase(words=4)   # ~44 Bits (Standard)
generate_passphrase(words=5)   # ~55 Bits
generate_passphrase(words=6)   # ~66 Bits

Benutzerdefiniertes Trennzeichen

Python
generate_passphrase(separator="-")   # correct-horse-battery-staple
generate_passphrase(separator=" ")   # correct horse battery staple
generate_passphrase(separator=".")   # correct.horse.battery.staple
generate_passphrase(separator="")    # correcthorsebatterystaple

Großschreibung

Python
generate_passphrase(capitalize=True, separator=" ")
# z.B. "Correct Horse Battery Staple"

Benutzerdefinierte Wortliste

Python
from password_generator import generate_passphrase, PassphraseConfig

config = PassphraseConfig(words=4, wordlist_path="/pfad/zur/wortliste.txt")
passphrase = generate_passphrase(config)

Entropie nach Wortanzahl

WörterEntropieBeispiel
2~22 Bitscorrect-horse
3~33 Bitscorrect-horse-battery
4~44 Bitscorrect-horse-battery-staple
6~66 Bitscorrect-horse-battery-staple-moon-river
8~88 Bits8 Wörter
10~110 Bits10 Wörter
Empfehlung: Verwenden Sie mindestens 4 Wörter (~44 Bits) für den allgemeinen Gebrauch oder 6+ Wörter (~66 Bits) für hochsichere Anwendungen.
