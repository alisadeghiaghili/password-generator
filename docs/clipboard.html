Clipboard Integration

Cross-platform clipboard support with automatic clearing. Copy passwords safely — they disappear from your clipboard after a configurable timeout.

Basic Usage

Python
from password_generator import generate, copy_to_clipboard

# Generate and copy
password = generate(length=24)
copy_to_clipboard(password)
print("Password copied to clipboard!")

Auto-Clear Clipboard

Automatically clear the clipboard after a timeout to prevent password exposure:

Python
from password_generator import generate, copy_to_clipboard

password = generate(length=24)

# Copy and auto-clear after 30 seconds
copy_to_clipboard(password, auto_clear_seconds=30)
print("Password copied! Clipboard clears in 30 seconds.")

# Or use a longer timeout
copy_to_clipboard(password, auto_clear_seconds=120)  # 2 minutes

Manual Clear

Python
from password_generator import clear_clipboard

# Clear clipboard immediately
clear_clipboard()
print("Clipboard cleared.")

Full Workflow

Python
from password_generator import generate, analyze, copy_to_clipboard

# Generate
password = generate(length=24)

# Analyze
report = analyze(password)
print(f"Strength: {report.score}/4")

# Copy with auto-clear
if report.score >= 3:
    copy_to_clipboard(password, auto_clear_seconds=30)
    print("Strong password copied! Clears in 30s.")
else:
    print("Password too weak — not copying.")
