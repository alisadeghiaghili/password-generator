CLI Guide

Interactive wizard for non-technical users and powerful command-line arguments for scripting and automation.

Interactive Mode

Run without arguments to start the guided wizard:

bash
python cli.py

The wizard guides you through:

  1. Choosing generation type (password / passphrase / PIN / analyze)
  2. Configuring options with simple yes/no prompts
  3. Viewing results with strength analysis
  4. Optionally copying to clipboard

Generate Passwords

bash
# Default: 16-character password
python cli.py --length 16

# 5 passwords of 20 characters
python cli.py --length 20 --count 5

# Uppercase + digits only
python cli.py --length 12 --no-lowercase --no-symbols

# Exclude ambiguous characters
python cli.py --length 20 --exclude-ambiguous

Generate Passphrases

bash
# 4-word passphrase (default)
python cli.py --passphrase

# 6 words, space-separated, capitalized
python cli.py --passphrase --words 6 --separator " " --capitalize

# 5 passphrases at once
python cli.py --passphrase --words 4 --count 5

Generate PINs

bash
# 4-digit PIN (default)
python cli.py --pin

# 6-digit PIN
python cli.py --pin --pin-length 6

# 8-digit, no repeats, no sequences
python cli.py --pin --pin-length 8 --avoid-repeats --avoid-sequential

# 5 backup codes
python cli.py --pin --pin-length 6 --avoid-repeats --count 5

Analyze Passwords

bash
# Analyze a password
python cli.py --analyze "MyP@ssw0rd"

# Analyze with JSON output
python cli.py --analyze "Tr0ub4dor&3" --json

JSON Output

For scripting and automation:

bash
# Generate as JSON
python cli.py --length 20 --json
# {
#   "type": "password",
#   "count": 1,
#   "passwords": ["k8#Qx!2mNp@Lw9Rz"]
# }

# Passphrase as JSON
python cli.py --passphrase --words 4 --json

# PIN as JSON
python cli.py --pin --pin-length 6 --json

Clipboard

bash
# Generate and copy to clipboard
python cli.py --length 24 --clipboard

# Copy with custom auto-clear (60 seconds)
python cli.py --length 24 --clipboard --clipboard-clear 60

Piping & Automation

bash
# Generate and extract with jq
python cli.py --length 16 --json | jq '.passwords[0]'

# Generate multiple and save to file
python cli.py --length 20 --count 10 --json > passwords.json

# Use in a script
DB_PASS=$(python cli.py --length 32 --no-symbols --json | jq -r '.passwords[0]')
