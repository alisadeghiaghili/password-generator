Welcome

A secure, configurable password generation toolkit for Python. Generate random passwords, XKCD-style passphrases, numeric PINs, and analyze password strength — all from a simple API or an interactive CLI.

🔐

Random Passwords

Cryptographically secure passwords with fully customizable character sets. Length from 4 to 256 characters.

🐴

Passphrases

XKCD-style memorable passphrases from a 2048-word list. "correct-horse-battery-staple" style.

🔢

PINs

Numeric codes with repeat and sequence avoidance. Perfect for 2FA backup codes.

📊

Strength Analysis

Entropy scoring, crack-time estimates, pattern detection. Know how strong your passwords really are.

📋

Clipboard Integration

Auto-copy with timed auto-clear. Cross-platform support for Windows, macOS, and Linux.

⌨️

Interactive CLI

Menu-driven wizard for non-technical users. JSON output for scripting and automation.

Quick Start

Python 
from password_generator import generate, generate_passphrase, generate_pin, analyze

# Generate a random 20-character password
password = generate(length=20)
print(password)  # e.g. "k8#Qx!2mNp@Lw9Rz"

# Generate a 5-word passphrase
passphrase = generate_passphrase(words=5, separator=" ", capitalize=True)
print(passphrase)  # e.g. "Correct Horse Battery Staple Moon"

# Generate a 6-digit PIN
pin = generate_pin(length=6, avoid_repeats=True, avoid_sequential=True)
print(pin)  # e.g. "849205"

# Analyze password strength
report = analyze("P@ssw0rd123")
print(f"Score: {report.score}/4")  # 0=Very Weak ... 4=Very Strong
Zero dependencies — Uses only Python's standard library (secrets, string, math). No external packages required.
