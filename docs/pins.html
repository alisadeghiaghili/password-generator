PINs

Generate random numeric PINs with optional repeat and sequence avoidance. Perfect for 2FA backup codes, device PINs, and verification codes.

Basic Usage

Python
from password_generator import generate_pin

# Default: 4-digit PIN
pin = generate_pin()
print(pin)  # e.g. "8492"

Custom Length

Python
# Standard 6-digit PIN
pin = generate_pin(length=6)

# 8-digit strong PIN
pin = generate_pin(length=8)

# Single digit (1-12 supported)
digit = generate_pin(length=1)

Avoid Repeated Digits

Prevent 3+ identical digits in a row (no "1111", "000999"):

Python
# No repeated digits
pin = generate_pin(length=6, avoid_repeats=True)
# Never: 111234, 999888, 000111

Avoid Sequential Digits

Prevent ascending or descending sequences (no "1234", "4321"):

Python
# No sequential digits
pin = generate_pin(length=4, avoid_sequential=True)
# Never: 1234, 4321, 5678, 8765

Combine Both Restrictions

Python
# Maximum security: no repeats AND no sequences
pin = generate_pin(
    length=6,
    avoid_repeats=True,
    avoid_sequential=True,
)

Using PinConfig

Python
from password_generator import generate_pin, PinConfig

# Reusable config for 2FA backup codes
backup_config = PinConfig(
    length=6,
    avoid_repeats=True,
    avoid_sequential=True,
)

# Generate 5 backup codes
for i in range(5):
    print(f"Code {i+1}: {generate_pin(backup_config)}")

Parameters
