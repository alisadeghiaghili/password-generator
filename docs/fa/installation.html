نصب

پیش‌نیازها

  • پایتون 3.10 یا بالاتر
  • بدون وابستگی خارجی — فقط کتابخانه استاندارد
  • اختیاری: rich برای خروجی زیبای CLI
  • کلیپ‌بورد لینوکس نیاز به xclip دارد

از PyPI

bash
# کتابخانه اصلی (بدون وابستگی)
pip install password-generator

# با خروجی زیبای CLI (رابط ترمینال ریچ)
pip install password-generator[cli]
نکته: افزونه [cli] کتابخانه rich را نصب می‌کند برای پنل‌های رنگی، نوارهای قوت و جدول‌های قالب‌بندی شده در ترمینال.

از سورس

bash
git clone https://github.com/alisadeghiaghili/password-generator.git
cd password-generator
pip install -e ".[cli]"

نصب توسعه

bash
pip install -e ".[dev,cli]"
pytest

بررسی

Python
from password_generator import generate
print(generate(length=16))
نکته: برای کلیپ‌بورد لینوکس: sudo apt install xclip
