پسورد تصادفی

تولید رمز عبور تصادفی رمزنگارانه ایمن با کنترل کامل روی مجموعه کاراکترها، طول و فیلتر ابهام.

استفاده پایه

Python
from password_generator import generate
password = generate()  # پیش‌فرض: 16 کاراکتر
print(password)

طول سفارشی

Python
generate(length=8)    # حداقل 4
generate(length=64)   # حداکثر 256

کنترل مجموعه کاراکترها

Python
generate(digits=False, symbols=False)        # فقط حروف
generate(symbols=False)                      # حروف و اعداد
generate(lowercase=False, symbols=False)     # حروف بزرگ + اعداد

نمادهای سفارشی

Python
generate(symbol_chars="!@#$%")
generate(symbol_chars="-_.~")                # امن برای شل

حذف کاراکترهای مبهم

Python
generate(length=20, exclude_ambiguous=True)  # بدون l, I, 1, O, 0, o

پارامترها

پارامترنوعپیش‌فرضتوضیح
lengthint16طول (4–256)
uppercaseboolTrueحروف بزرگ
lowercaseboolTrueحروف کوچک
digitsboolTrueاعداد
symbolsboolTrueنمادها
exclude_ambiguousboolFalseحذف کاراکترهای مبهم

نحوه کار

  1. بذرگذاری — یک کاراکتر از هر مجموعه فعال
  2. پرکردن — بقیه از مجموعه ترکیبی
  3. .Shuffle — فیشر-یتس رمزنگارانه
امنیت: از ماژول secrets استفاده می‌کند. هرگز از random استفاده نکنید.
