کد PIN

تولید کد PIN عددی تصادفی با جلوگیری اختیاری از تکرار و توالی.

استفاده پایه

Python
from password_generator import generate_pin
pin = generate_pin()  # 4 رقمی
print(pin)

طول سفارشی

Python
generate_pin(length=6)
generate_pin(length=8)

جلوگیری از تکرار

Python
generate_pin(length=6, avoid_repeats=True)
# هیچوقت: 111234, 999888

جلوگیری از توالی

Python
generate_pin(length=4, avoid_sequential=True)
# هیچوقت: 1234, 4321

پارامترها

پارامترنوعپیش‌فرضتوضیح
lengthint4طول (1–12)
avoid_repeatsboolFalseجلوگیری از تکرار
avoid_sequentialboolFalseجلوگیری از توالی
