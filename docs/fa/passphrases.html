عبارت عبور

تولید عبارات عبور XKCD از لیست 2048 کلمه‌ای. به یادماندنی و امن.

استفاده پایه

Python
from password_generator import generate_passphrase
phrase = generate_passphrase()  # 4 کلمه، خط تیره
print(phrase)  # مثلاً: "correct-horse-battery-staple"

تعداد کلمات

Python
generate_passphrase(words=3)   # ~33 بیت
generate_passphrase(words=4)   # ~44 بیت (پیش‌فرض)
generate_passphrase(words=6)   # ~66 بیت

جداکننده سفارشی

Python
generate_passphrase(separator=" ")   # فاصله
generate_passphrase(separator=".")   # نقطه
generate_passphrase(separator="")    # بدون جداکننده

بزرگ‌نویسی

Python
generate_passphrase(capitalize=True, separator=" ")

لیست کلمات سفارشی

Python
from password_generator import generate_passphrase, PassphraseConfig
config = PassphraseConfig(words=4, wordlist_path="wordlist.txt")
print(generate_passphrase(config))

آنتروپی

کلماتآنتروپیمثال
2~22 بیتcorrect-horse
4~44 بیتcorrect-horse-battery-staple
6~66 بیت6 کلمه
10~110 بیت10 کلمه
توصیه: حداقل 4 کلمه (~44 بیت) برای استفاده عمومی، 6+ کلمه (~66 بیت) برای امنیت بالا.
