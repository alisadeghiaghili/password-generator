تحلیل قوت

تحلیل قوت رمز عبور با امتیازدهی آنتروپی، برآورد زمان شکستن و تشخیص الگو.

استفاده پایه

Python
from password_generator import analyze
report = analyze("MyP@ssw0rd123!")
print(f"امتیاز: {report.score}/4")
print(f"آنتروپی: {report.entropy:.0f} بیت")

Python
report = analyze("k8#Qx!2mNp@Lw9Rz")
