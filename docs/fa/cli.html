راهنمای CLI

راهنمای تعاملی و آرگومان‌های خط فرمان برای اسکریپت‌نویسی و اتوماسیون. با نصب rich، CLI خروجی پنل‌های رنگی، نوارهای قوت و جدول‌های قالب‌بندی شده نمایش می‌دهد.

CLI ریچ: نصب با pip install password-generator[cli] برای خروجی زیای ترمینال با پنل‌های رنگی، نوارهای قوت و جدول‌های قالب‌بندی شده.

حالت تعاملی

bash
python cli.py

تولید پسورد

bash
python cli.py --length 16
python cli.py --length 20 --count 5
python cli.py --length 12 --no-lowercase --no-symbols
terminal — password-gen
┌──────────────────── Generated 3 password(s) (20 chars) ─────────────────────┐ │ │ │ 1 NA||v?S![2 │ │ 2 8<As5}pY<P$%Bv2Yz1Ze │ │ 3 uIQ1@cA6GgxW}_,xg3R+ │ │ │ └─────────────────────────────────────────────────────────────────────────────┘

تولید عبارت عبور

bash
python cli.py --passphrase
python cli.py --passphrase --words 6 --separator " " --capitalize
terminal — password-gen --passphrase
┌──────────────────── Generated 1 passphrase(s) (4 words) ────────────────────┐ │ │ │ 1 opera-macro-speak-pinch │ │ │ └─────────────────────────────────────────────────────────────────────────────┘

تولید PIN

bash
python cli.py --pin
python cli.py --pin --pin-length 6 --avoid-repeats --avoid-sequential
