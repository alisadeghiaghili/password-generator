کلیپ‌بورد

پشتیبانی کلیپ‌بورد چندپلتفرمی با پاکسازی خودکار.

استفاده پایه

Python
from password_generator import generate, copy_to_clipboard
password = generate(length=24)
copy_to_clipboard(password)

پاکسازی خودکار

Python
copy_to_clipboard(password, auto_clear_seconds=30)
copy_to_clipboard(password, auto_clear_seconds=120)

پاکسازی دستی

Python
from password_generator import clear_clipboard
clear_clipboard()
