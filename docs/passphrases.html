Passphrases

Generate memorable XKCD-style passphrases from a 2048-word list. Easy to remember, hard to crack.

Basic Usage

Python
from password_generator import generate_passphrase

# Default: 4 words, hyphen-separated
passphrase = generate_passphrase()
print(passphrase)  # e.g. "correct-horse-battery-staple"

Custom Word Count

Python
# More words = more entropy
three = generate_passphrase(words=3)   # ~33 bits
four = generate_passphrase(words=4)    # ~44 bits (default)
five = generate_passphrase(words=5)    # ~55 bits
six = generate_passphrase(words=6)     # ~66 bits

Custom Separator

Python
# Hyphen (default)
generate_passphrase(separator="-")      # correct-horse-battery-staple

# Space
generate_passphrase(separator=" ")      # correct horse battery staple

# Period
generate_passphrase(separator=".")      # correct.horse.battery.staple

# Underscore
generate_passphrase(separator="_")      # correct_horse_battery_staple

# No separator
generate_passphrase(separator="")       # correcthorsebatterystaple

Capitalization

Python
# Capitalize first letter of each word
passphrase = generate_passphrase(capitalize=True, separator=" ")
print(passphrase)  # e.g. "Correct Horse Battery Staple"

Custom Wordlist

Use your own word list (minimum 100 words):

Python
from password_generator import generate_passphrase, PassphraseConfig

config = PassphraseConfig(
    words=4,
    separator="-",
    wordlist_path="/path/to/my/wordlist.txt"
)
passphrase = generate_passphrase(config)

Your wordlist file should have one word per line:

wordlist.txt
apple
banana
cherry
delta
echo
...

Using PassphraseConfig

Python
from password_generator import generate_passphrase, PassphraseConfig

# Reusable config
config = PassphraseConfig(
    words=6,
    separator=" ",
    capitalize=True,
)

for i in range(3):
    print(generate_passphrase(config))
# Correct Horse Battery Staple Moon River
# Lucky Forest Jump Dark Ocean Stone
# ...

Parameters

ParameterTypeDefaultDescription
wordsint4Number of words (2–10)
separatorstr-Word separator
capitalizeboolFalseCapitalize first letter
wordlist_pathstrNoneCustom wordlist file (min 100 words)
