Installation

Requirements

  • Python 3.10 or higher
  • No external dependencies — uses only Python's standard library
  • Linux clipboard requires xclip or xsel

From PyPI

The easiest way to install:

bash
pip install password-generator

From Source

Install directly from GitHub:

bash
git clone https://github.com/alisadeghiaghili/password-generator.git
cd password-generator
pip install -e .

Development Install

For contributing or running tests:

bash
# Clone and enter the repo
git clone https://github.com/alisadeghiaghili/password-generator.git
cd password-generator

# Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # macOS/Linux
venv\Scripts\activate     # Windows

# Install with dev dependencies
pip install -e ".[dev]"

# Verify installation
pytest

Verify It Works

Python
from password_generator import generate
print(generate(length=16))  # e.g. "k8#Qx!2mNp@Lw9Rz"

CLI Verification

bash
# Interactive mode
python cli.py

# Quick generation
python cli.py --length 16 --count 3
Tip: If you plan to use clipboard features on Linux, install xclip: sudo apt install xclip (Debian/Ubuntu) or sudo dnf install xclip (Fedora).
