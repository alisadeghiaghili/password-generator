Strength Analysis

Analyze password strength with entropy scoring, crack-time estimates, and pattern detection. Know exactly how strong your passwords are.

Basic Usage

Python
from password_generator import analyze

report = analyze("MyP@ssw0rd123!")
print(f"Score: {report.score}/4")       # 0=Very Weak ... 4=Very Strong
print(f"Entropy: {report.entropy:.0f} bits")
print(f"Feedback: {report.feedback}")

Understanding the Score

Crack Time Estimates

The analyzer estimates crack times for four attack scenarios:

Python
report = analyze("correct-horse-battery-staple")

for scenario, time_str in report.crack_times.items():
    print(f"{scenario}: {time_str}")

# Output:
# online_throttled_100_per_hour: centuries
# online_no_throttling_10_per_second: decades
# offline_slow_hashing_1e4_per_second: hours
# offline_fast_hashing_1e10_per_second: seconds

Attack Scenarios

ScenarioRateDescription
online_throttled100/hourLogin with rate limiting
online_no_throttling10/secondLogin without rate limiting
offline_slow_hashing10K/secondStolen hash, slow algorithm (bcrypt)
offline_fast_hashing10B/secondStolen hash, fast algorithm (MD5/SHA)

Pattern Detection

The analyzer detects five types of weak patterns:

Python
# Common passwords
report = analyze("password")
print(report.patterns)  # ["common_password"]

# Keyboard patterns
report = analyze("qwerty")
print(report.patterns)  # ["keyboard_pattern"]

# Sequences
report = analyze("abcdef")
print(report.patterns)  # ["sequence"]

# Repeated characters
report = analyze("aaa111")
print(report.patterns)  # ["repetition"]

# Date patterns
report = analyze("2024-01-15")
print(report.patterns)  # ["date"]

All Detected Patterns

Feedback

Actionable feedback to help users improve their passwords:

Python
report = analyze("abc")
for feedback in report.feedback:
    print(f"- {feedback}")
# - Contains sequential characters (e.g., abc, 123)
# - Password is too short (minimum 8 characters)

report = analyze("Tr0ub4dor&3")
print(report.feedback)  # ["Good password!"]

Full Report

Python
report = analyze("k8#Qx!2mNp@Lw9Rz")

# All available fields
print(report.password)              # "****************" (masked)
print(report.score)                 # 4
print(report.entropy)               # ~106
print(report.guesses)               # ~8.5e+31
print(report.crack_times)           # dict of scenario → human-readable time
print(report.crack_times_seconds)   # dict of scenario → seconds
print(report.feedback)              # ["Good password!"]
print(report.patterns)              # [] (no weak patterns)
Security note: The password is masked in the report (shown as asterisks) to prevent accidental exposure in logs.
