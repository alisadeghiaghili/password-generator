generate()

Generate a cryptographically secure random password.

Python
def generate(config: GeneratorConfig | None = None, **kwargs) -> str

Parameters

ParameterTypeDefaultDescription
configGeneratorConfigNoneConfiguration object (overrides kwargs)
lengthint16Password length (4–256)
uppercaseboolTrueInclude A–Z
lowercaseboolTrueInclude a–z
digitsboolTrueInclude 0–9
symbolsboolTrueInclude symbols
symbol_charsstr!@#$%^&*…Custom symbol characters
exclude_ambiguousboolFalseExclude l, I, 1, O, 0, o

Returns

str — A random password string of the specified length.

Raises

  • ValueError — If length is outside 4–256
  • ValueError — If no character categories are enabled
  • ValueError — If length is too short for enabled categories

Examples

Python
generate()                                          # 16 chars, all types
generate(length=24)                                 # 24 chars
generate(length=12, symbols=False)                  # No symbols
generate(GeneratorConfig(length=32, exclude_ambiguous=True))

Parameters

ParameterTypeDefaultDescription

separatorstr-Word separator
capitalizeboolFalseCapitalize first letter
wordlist_pathstrNoneCustom wordlist (min 100 words)

Returns

str — A passphrase string like "correct-horse-battery-staple".

Examples

Python

generate_passphrase(words=6, separator=" ")          # 6 words, space
generate_passphrase(capitalize=True)                 # Capitalized
